Web Design

Web design diffrent skills and disciplines in the production and maintenence of websites. The different areas of web design include web graphic design; user interface design (UI design); authoring, including standardised code and proprietary software;user experience design (UX design); and search engine optimization. Often many individuals will work in teams covering defferent aspects of the design proces, although some designers will cover them all. The term "web design" is normally used to describe the design proces relating to the front-end (client side) design of a website including writing markup. Web design partially overlaps engineering in the broader scope of web development. Web designers are expected to have an awarness of usability and be up to date with web accessbility guidlines.
